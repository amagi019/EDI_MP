# EDI-MP パートナー向け操作マニュアル

EDIシステム（EDI-MP）をご利用いただきありがとうございます。
本マニュアルでは、パートナー（取引先）様がシステムをスムーズにご利用いただくための手順を説明します。

---

## 1. 初回ログインとパスワード変更

1. **ログイン**: 管理者から発行されたIDと仮パスワードでログインしてください。
2. **パスワード変更**: 初回ログイン時は、セキュリティのためパスワード変更画面が表示されます。新しいパスワードを設定してください。

## 2. 会社情報の登録・更新（オンボーディング）

システムを正しく利用し、迅速な支払いを受けるために、貴社の基本情報を登録してください。

1. **画面へのアクセス**: ダッシュボードの「会社情報を登録・更新する」ボタンをクリックします。
2. **入力項目**:
   - 会社名、フリガナ、住所、代表者情報。
   - **インボイス登録番号**: Tから始まる13桁の番号を入力してください。
   - **振込先情報**: 正確な支払いのために、金融機関名、支店名、口座情報を入力してください。
3. **保存**: 「登録完了」をクリックして保存します。

## 3. 基本契約書の確認と承諾

1. **通知**: 基本契約書が作成されると、メールで通知が届きます。
2. **閲覧**: メール内のURLからログインし、契約書の内容を確認します。
3. **承諾**: 内容に問題がなければ「承諾」ボタンをクリックしてください。承諾完了後、契約が締結されます。
4. **確認**: 承諾済みの契約書は、サイドバーの「基本契約書」からいつでも確認できます。

> **法的根拠について**: 本システムの契約締結は、電子契約法（電子消費者契約及び電子承諾通知に関する民法の特例に関する法律）および電子帳簿保存法に基づき、電子署名がなくても法的に有効です。当事者間の合意をシステム上で記録し、タイムスタンプとともに保存しています。ご要望に応じて、電子署名サービスを利用した契約手続きも可能です。

## 4. 注文書の確認と承諾

1. **通知**: 新しい注文書が発行されると、メールで通知が届きます。
2. **閲覧**: メール内のURLまたはダッシュボードの「注文書一覧」から対象の注文を選択し、詳細を確認します。「注文書印刷（PDF）」をクリックして内容を確認してください。
3. **承諾**: 内容に問題がなければ「承諾」ボタンをクリックしてください。
4. **注文請書**: 承諾が完了すると、「注文請書」がシステムにより自動生成され、エビデンスとして保存されます。

## 5. 稼働報告書の提出

毎月の稼働実績をExcelファイルでアップロードしてください。月末付近にメールでリマインド通知が届きます。

1. **画面へのアクセス**: サイドバーの「稼働報告書」をクリックします。
2. **注文の選択**: 対象の注文をドロップダウンから選択してください。
3. **ファイルのアップロード**: Excel形式（.xlsx / .xlsm）の稼働報告書をアップロードします。複数ファイルの同時アップロードにも対応しています。
4. **結果の確認**: アップロード後、自動で氏名・稼働時間・土日祝稼働が解析されます。警告がある場合は内容を確認してください。
5. **確定**: 内容に問題がなければ「稼働報告を確定」ボタンをクリックしてください。確定後、弊社担当者に通知されます。

## 6. 支払通知の確認

1. **通知**: 支払通知書が発行されると、メールで通知が届きます。
2. **確認**: メニューの「請求書（支払通知）一覧」から支払金額の内容を確認してください。
3. **承諾**: 内容に問題がなければ「承諾」をクリックしてください。金額に相違がある場合は、弊社担当者までご連絡ください。
