# メールテンプレート（支払通知書・請求書送付）

## 件名

【EDI-MP】支払通知書・請求書のご送付（{対象年月}分）

## 本文

```
{パートナー名} 御中

いつもお世話になっております。
有限会社マックプランニングでございます。

EDI-MPに支払通知書を登録しましたので、
下記ＵＲＬからユーザＩＤ／パスワードを用いてログインし、
請求書詳細画面から請求書ファイル（ＰＤＦ）のダウンロードを行って下さい。
ダウンロードした請求書は、貴社にて必ず保管して下さい。

《URL》
{請求書詳細URL}

《送付物》
「支払通知書」
「請求書」
  各1通

《お願い》
請求書内容をご確認いただき、内容にご同意いただける場合は「承認」ボタンを押してください。
承認いただけない場合は本メールにご返信ください。

   締切：{支払締切日}

※尚、「承認」をいただいた場合は、従来のように請求書のご返送は不要となります。

御手数ではございますが、ご協力の程お願い申し上げます。


操作手順について不明な点がございましたら
EDI-MPの「操作マニュアル」をご参考ください。

以上、よろしくお願いします。
```

## 変数一覧

| 変数 | 説明 | 例 |
|------|------|-----|
| `{パートナー名}` | 送付先パートナーの会社名 | 株式会社○○○ |
| `{対象年月}` | 請求対象の年月 | 2026年3月 |
| `{請求書詳細URL}` | EDI-MPの請求書詳細画面URL | https://edi.macplanning.com/invoices/my/123/ |
| `{支払締切日}` | 支払の締切日 | 2026年3月20日 |
