# 多言語化 ウォークスルー

アプリケーションのヘッダーとフッターを日本語化しました。

## 変更点

### `core/templates/base.html`
- ヘッダーのハードコードされた英語 "EDI SYSTEM" を `{% trans "EDIシステム" %}` に置き換えました。
- フッターのハードコードされた英語 "EDI System" を `{% trans "EDIシステム" %}` に置き換えました。

## 検証
Python 環境のセットアップが完了しました。以下の手順で検証を行ってください。

### 検証手順:
1.  **サーバーの起動**:
    プロジェクトフォルダで以下のコマンドを実行してください（仮想環境の `python` を使用）。
    ```powershell
    & ".\.venv\Scripts\python.exe" manage.py runserver
    ```
    ※ ターミナル再起動後であれば `.venv\Scripts\activate` してから `python manage.py runserver` でも可能です。

2.  **ブラウザ確認**:
    ブラウザで `http://127.0.0.1:8000/` を開きます。

3.  **表示確認**:
    - ヘッダーの左上が "EDI SYSTEM" ではなく "**EDIシステム**" と表示されていることを確認してください。
    - フッターのコピーライトが "**EDIシステム**" と表示されていることを確認してください。
