# 協力会社向け リモートデプロイ手順書

## 概要
協力会社のNASにEDIアプリをリモートで構築する手順です。
先方にIT知識は不要です。

---

## 事前準備（御社）

- [ ] ソースコード一式を用意（Git clone → ZIP化）
- [ ] `.env.nas.sample` を同梱済みか確認
- [ ] `setup.sh` を同梱済みか確認
- [ ] Tailscaleアカウントを準備
- [ ] Zoom会議を設定

---

## 当日の流れ（所要時間：約1時間）

### フェーズ1：Zoom画面共有（約15分）

先方にやってもらうこと（御社が画面を見ながら指示）：

- [ ] ① Zoom会議に参加してもらう
- [ ] ② ブラウザでNAS管理画面を開いてもらう
  - `http://NASのIPアドレス:5000`（Synologyの場合）
- [ ] ③ 画面共有してもらう
- [ ] ④ 以下を御社が指示して先方にクリックしてもらう：
  - **コントロールパネル → ターミナルとSNMP → SSHサービスを有効化**
  - **パッケージセンター → Docker → インストール**
  - **パッケージセンター → Tailscale → インストール → ログイン**
- [ ] ⑤ NASのIPアドレスを教えてもらう（コントロールパネル → ネットワーク）

### フェーズ2：リモート構築（約30分）

SSHで先方のNASに接続後、御社が実施：

- [ ] ⑥ Tailscale経由でSSH接続
  ```bash
  ssh admin@100.x.x.x
  ```
- [ ] ⑦ ソースコードの転送
  ```bash
  scp -r EDI_MP.zip admin@100.x.x.x:/volume1/docker/
  ```
- [ ] ⑧ セットアップスクリプト実行
  ```bash
  cd /volume1/docker/EDI_MP
  bash setup.sh
  ```
  - Antigravityに「セットアップして」と依頼すれば支援可能
- [ ] ⑨ 動作確認（ブラウザでアクセス）

### フェーズ3：初期設定案内（約15分）

Zoomで先方に案内：

- [ ] ⑩ ブラウザでアプリにアクセスしてもらう
  - `http://NASのIP:8090/admin/`
- [ ] ⑪ 管理画面で会社情報を入力してもらう
  - 会社名、住所、代表者名
  - 印影画像のアップロード
- [ ] ⑫ 動作テスト（パートナー登録 → メール送信確認）

---

## 完了後

- 先方は日常的に `http://NASのIP:8090/` にアクセスするだけ
- メンテナンスが必要な場合はTailscale経由でリモート対応
- アプリの更新も `git pull && docker compose restart` のみ
