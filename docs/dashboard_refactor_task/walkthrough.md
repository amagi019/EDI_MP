# ダッシュボード改修＆不具合修正 ウォークスルー

ダッシュボードの表示整理と、基本契約進捗一覧画面の表示不具合を修正しました。

## 変更内容

### 1. ダッシュボード (`dashboard.html`)
- **進捗リストの削除**: 画面下部に表示されていた「取引先オンボーディング・基本契約進捗」のテーブルを削除しました。
- **ボタン修正**: 「取引先新規登録」ボタンのスタイルを、他のボタン（注文書一覧など）と同じグレーのスタイル（`btn-secondary`）に変更しました。

### 2. 基本契約進捗一覧 (`contract_progress_list.html`)
- **表示不具合の修正**: テンプレート内で使用されていたCSS変数が `base.html` の定義と不整合を起こしていたため、正しい変数名（`--border`, `--surface` 等）に修正し、正しく表示されるように再実装しました。

## 検証手順

1.  **サーバーの起動**:
    ```powershell
    & ".\.venv\Scripts\python.exe" manage.py runserver
    ```

2.  **ダッシュボード確認**:
    - 「取引先オンボーディング・基本契約進捗」のテーブルが表示されていないことを確認してください。
    - 「取引先新規登録」ボタンがグレーのデザインに変更されていることを確認してください。

3.  **進捗一覧画面の確認**:
    - ダッシュボードの「基本契約進捗」ボタンをクリックしてください。
    - 「基本契約進捗一覧」画面がエラーなく表示され、テーブルのデザインが崩れていないことを確認してください。
